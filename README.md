# ReFormAI 🏗️

**AI-Powered Automation of Formwork Kitting**

🚀 **Live Demo:** 👉 [https://github.com/SonaaRajagopal/ReFormAI-Automation-of-Formwork-Kitting/](https://github.com/SonaaRajagopal/ReFormAI-Automation-of-Formwork-Kitting/)

ReFormAI is an intelligent formwork planning and kitting platform that automates **BoQ optimization, reuse planning, and cost-efficient kit generation** by integrating **BIM models, project schedules, and inventory data**.

Built to replace manual, spreadsheet-driven planning with **AI-led, predictive decision-making**.

---

## 🚧 Problem

Formwork accounts for **7–10% of total construction cost**, yet planning today is:

* Manual and spreadsheet-based
* Static BoQ with poor demand accuracy
* No systematic reuse or repetition detection
* Disconnected from BIM and inventory systems

This results in:

* Excess inventory and shortages
* Procurement delays and cost overruns
* Low formwork lifecycle utilization

---

## 💡 Solution — ReFormAI

ReFormAI introduces an **end-to-end AI pipeline** that automates formwork planning:

1. **BIM-Based Repetition Intelligence**
   Detects similar structural elements using clustering to maximize reuse across floors and phases.

2. **Phase-Wise Demand Forecasting**
   Time-series models predict material requirements aligned to project schedules.

3. **Mathematical Cost Optimization**
   Linear programming minimizes procurement and inventory holding costs under reuse and schedule constraints.

4. **Smart Kit Generation**
   Automatically generates ready-to-deploy, phase-specific formwork kits.
